UST IICS ALUMNI ASSOCIATION

And so it begins...

Hello world! This is my first commit to the UST IICS ALUMNI ASSOCIATION PROJECT. I hope to open this project up to future IICS Alumni and students, so that we get to have a community that's centered around collaboration and open-source projects. Through this project I hope that we will learn and grow together in our alumni group.
The project should always be maintained with the users in mind. Top priority is to share information and data in the most efficient way through modern and stable resources. If we can find simpler ways of implementation for our objectives we should prefer them over complex methods. That is the reason for starting with a simple html5 and css boilerplate, so we can drive this project forward in an agile manner and put out the needs of the association and it's members. Let's focus on the user's needs and the project's utility above all. With peace and love to all - JMA.
